## Математика для Data Science
### Статистика
#### Основные определения, необходимые для понимания материала
##### Генеральная совокупность  – группа объектов, которые вам интересны как исследователю.
##### Выборка  – часть генеральной совокупности, доступная для исследования. Статистики стремятся к тому, чтобы результаты, полученные на выборках, были верны и для генеральной совокупности.
##### Связанные выборки – ситуация, при которой любому объекту из первой выборки соответствует ровно один объект из второй. Можно сказать, что они образуют неразрывную пару (а в более сложных случаях – тройку, четверку и т. д.).
##### Наблюдение – измеренный объект.
##### Переменные – свойства объектов, которые поддаются измерению.
##### Значение переменной  – степень выраженности того или иного свойства у конкретного объекта.
